# 03. 화면 구성 및 와이어프레임

## 개요

MapVlog 앱의 주요 화면 구조와 레이아웃을 텍스트 기반 와이어프레임으로 정의합니다.
실제 UI 구현 시 이 문서를 기준으로 Flutter 위젯을 구성합니다.

---

## 앱 네비게이션 구조

```
┌─────────────────────────────────────────────┐
│                  앱 전체 구조                  │
│                                             │
│  스플래시 → 온보딩 → 로그인                    │
│                  │                          │
│                  ▼                          │
│         ┌────────────────┐                  │
│         │   메인 탭바     │                  │
│         │  홈 | 지도 |   │                  │
│         │  촬영 | 갤러리 |│                  │
│         │   프로필       │                  │
│         └────────────────┘                  │
└─────────────────────────────────────────────┘
```

## 컴포넌트 정의

| 컴포넌트 | 설명 | 재사용 화면 |
|---------|------|------------|
| VlogCard | 썸네일+제목+위치+시간 카드 | 홈, 프로필 |
| MapMarker | 브이로그 위치 마커 | 지도, 갤러리 |
| PhotoMarker | 사진 미니 썸네일 마커 | 지도, 갤러리 |
| GpsStatusBar | GPS 좌표 + 위성 수신 상태 | 촬영 |
| VideoPlayerWidget | 영상 + 진행바 + 컨트롤 | 플레이어 |
| PlaceInfoCard | 현재 위치 장소 정보 팝업 | 플레이어, 촬영 |
| BottomTabBar | 하단 탭 네비게이션 | 전체 |
